##  À propos de RevLearn
RevLearn est une plateforme éducative ultra-rapide et gratuite. 
Conçue avec un design hautement professionnel (inspiré des documentations techniques les plus prestigieuses de l'industrie), la plateforme offre un confort de lecture exceptionnel pour apprendre le développement logiciel, le tout sans framework lourd ni base de données complexe.

##  Fonctionnalités & Contenu
- **Interface Pro** : Mode sobre, polices typographiques dédiées au code (`JetBrains Mono`).
- **Maîtrise de Git** : Comprendre et dompter l'historique et les branches.
- **Terminal & Bash** : Apprendre à contrôler sa machine via le shell Unix.
- **Fondations HTML/CSS** : Créer et structurer des pages web responsives.
- **JavaScript Moderne** : Manipuler le DOM et maîtriser ES6+.

##  Installation Rapide
La plateforme est constituée de fichiers statiques HTML/CSS/JS simples et ne nécessite aucune configuration.

1. Clonez ce dépôt sur votre machine :
   ```bash
   git clone https://github.com/AnARCHIS12/learn-gity.git
   ```
2. Ouvrez simplement le fichier `index.html` dans le navigateur web de votre choix :
   ```bash
   cd learn-gity
   # Sous linux ou macOS :
   xdg-open index.html 
   # ou open index.html
   ```

##  Conception Artistique
L'application se concentre sur les bonnes pratiques de conception UI :
- **Clarté Absolue** : Aucun élément distrayant, bordures très fines (`1px solid #e5e7eb`), ombres à peine perceptibles.
- **La Performance** : Utilisation de variables CSS natives et grilles fluides sans librairies tierces.

##  Contribution
Ce projet est open-source ! Vous pouvez ajouter de nouveaux cours en clonant par exemple le fichier `cours-git.html` et en utilisant ses classes.
