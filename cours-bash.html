Maîtrise du Terminal & Bash

Apprenez à communiquer directement avec votre système d'exploitation. Fini l'interface graphique : devenez un utilisateur avancé en exploitant la puissance brute du shell Unix.

Se repérer et naviguer

Le terminal fonctionne sous forme d'arborescence. À chaque instant, vous vous trouvez dans un dossier spécifique (votre "répertoire de travail actuel").

pwd

Print Working Directory : Permet de savoir exactement dans quel dossier complet vous vous trouvez actuellement.

ls

List : Affiche le contenu du dossier actuel. L'option -l donne les détails, et -a montre les fichiers cachés.

cd

Change Directory : Permet de vous déplacer, de rentrer dans un dossier ou d'en sortir avec cd ...

clear

Nettoie l'écran de votre terminal lorsque celui-ci est trop encombré.

# Je vérifie où je suis sur ma machine
pwd

# Je liste tout le contenu de mon dossier avec les détails
ls -la

# Je rentre dans le dossier "Documents"
cd Documents/

Fichiers et dossiers

Une fois que vous savez naviguer, il est temps d'apprendre à créer, déplacer, et supprimer des documents sans ouvrir le moindre gestionnaire de fenêtre.

1

Créer un élément

mkdir nom_du_dossier pour créer un répertoire. touch fichier.txt pour créer un simple fichier vide instantanément.

2

Copier ou déplacer

cp source destination pour copier, mv source destination pour déplacer. Attention : mv sert aussi à renommer les fichiers !

3

Supprimer définitivement

rm fichier.txt supprime un fichier. Pour un dossier entier avec tout son contenu, on utilisera méticuleusement rm -rf nom_du_dossier.

# Création rapide de l'arborescence
mkdir -p projet/src
touch projet/src/index.js

# Déplacement et renommage
mv projet/src/index.js projet/src/app.js

Recherche et lecture d'informations

Gérer de grandes quantités de textes et de données depuis le terminal est infiniment plus rapide si vous maîtrisez ces outils.

cat

Concatène et affiche le contenu intégral d'un fichier directement dans la fenêtre de votre terminal.

grep

Le puissant outil de recherche. Permet de trouver un mot spécifique ou un motif à l'intérieur de dizaines de fichiers instantanément.

# Affiche tout le contenu de mon fichier de logs
cat server.log

# Recherche toutes les lignes qui contiennent le mot "ERROR"
grep "ERROR" server.log

Entraînement

Validation des acquis par la pratique, tapez ceci de A à Z :

  1. Ouvrez votre console à l'emplacement de votre choix.
  2. Créez un dossier laboratoire : mkdir labo_bash.
  3. Entrez dans ce dossier : cd labo_bash.
  4. Générez un fichier vide : touch test.txt.
  5. Faites-en une copie de sécurité : cp test.txt test_backup.txt.
  6. Affichez le contenu du dossier pour vérifier vos actions : ls -l.

Architecture et documentation rédigée dans un standard professionnel.
