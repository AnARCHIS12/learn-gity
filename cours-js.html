JavaScript Moderne (ES6+)

Donnez vie à vos interfaces. JavaScript est le langage de programmation incontournable du navigateur, permettant de gérer l'interactivité, de contacter des serveurs et d'animer vos pages.

Variables et Constantes

En JavaScript moderne (ES6+), l'utilisation du mot-clé var est fortement déconseillée afin d'éviter les effets de bord indésirables.

const

Déclare une variable dont la référence ne changera jamais. C'est l'option recommandée par défaut 95% du temps.

let

Déclare une variable qui sera modifiée ou réaffectée plus tard dans l'exécution de votre script (ex: compteurs de boucle).

// Bonne pratique (ES6+)
const prenom = "Alice";
let age = 25;
age = 26; // Valide
// prenom = "Bob"; // Jeterait une erreur bloquante

Déclaration de fonctions

Les fonctions encadrent votre logique. Avec l'arrivée des "fonctions fléchées" (arrow functions), le langage s'est considérablement épuré.

// Déclaration standard (historique)
function saluer(nom) {
  return "Bonjour " + nom;
}

// Déclaration moderne (Arrow function avec template littéral)
const saluerModerne = (nom) => {
  return `Bonjour ${nom}`;
};

// L'arrow function permet même des retours implicites :
const multiplier = (a, b) => a * b;

Manipulation du DOM

Le Document Object Model (DOM) est la représentation HTML chargée par le navigateur. JavaScript permet d'interagir avec et de la modifier en direct.

Sélectionner

Utilisez l'omniprésent document.querySelector('.ma-classe') pour récupérer le premier élément qui correspond à un sélecteur CSS précis.

Agir

Manipulez ses classes, son contenu texte, ou ajoutez des écouteurs d'événements pour réagir aux clics avec addEventListener.

// Cible le bouton dans mon HTML
const bouton = document.querySelector('.action-btn');

// Je déclenche un code quand l'utilisateur clique dessus
bouton.addEventListener('click', () => {
  document.body.classList.toggle('dark-mode');
});

Entraînement

Mettez en pratique vos connaissances JavaScript :

  1. Dans un fichier HTML, créez un bouton "Afficher Alerte".
  2. Dans une balise <script> en fin de fichier, récupérez ce bouton via un sélecteur.
  3. Attachez-y un événement "click".
  4. Faites en sorte qu'une modale s'affiche avec alert("Cliqué !") au moment de l'action.
