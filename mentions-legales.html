1. Édition du projet

RevLearn est présenté ici comme un projet open source distribué sous forme de fichiers statiques. Le dépôt source fourni pour ce site est accessible à l'adresse github.com/AnARCHIS12/learn-gity.

Le propriétaire visible du dépôt au moment de la rédaction est le compte GitHub AnARCHIS12. Si vous exploitez ce site publiquement sous votre propre nom, il vous revient de compléter cette page avec les informations d'édition et de contact adaptées à votre situation.

2. Hébergement

Ce projet peut être déployé de différentes manières. Lorsqu'il est publié depuis l'URL renseignée dans le dépôt GitHub, il est raisonnable d'en déduire qu'une mise en ligne via GitHub Pages peut être utilisée. Si vous changez d'hébergeur, cette information doit être adaptée.

Le code source local que vous utilisez ici ne contient pas, à lui seul, la preuve définitive du mode d'hébergement actuellement actif. Cette page reste donc volontairement prudente sur ce point.

3. Propriété intellectuelle

Le code, le design et les contenus de cours présents dans ce projet restent soumis aux conditions choisies par l'auteur ou les auteurs du dépôt. Aucun fichier de licence explicite n'est présent à la racine du projet local au moment de cette rédaction.

En pratique, si vous souhaitez réutiliser largement le contenu, republier des cours ou redistribuer le site, il est préférable d'obtenir l'accord du propriétaire du dépôt ou d'ajouter une licence claire au projet avant diffusion.

4. Liens externes et services utilisés

Le site peut charger des ressources externes comme Bunny Fonts, cdnjs pour les icônes Font Awesome et GitHub pour le dépôt ou le suivi des problèmes. Ces services disposent de leurs propres conditions d'utilisation et politiques de confidentialité.

Lorsque vous cliquez sur un lien externe, vous quittez le périmètre du site RevLearn et vous dépendez alors des règles du service visité.

5. Contact

Le moyen de contact le plus direct renseigné dans ce projet est le dépôt GitHub: ouvrir une issue.

Si vous transformez ce site en publication publique durable, ajoutez ici vos informations de contact réelles et les éléments légaux requis par votre cadre de diffusion.
