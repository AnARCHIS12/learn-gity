Version 2.0.0 (Mars 2026)

Support de l'application hors-ligne (PWA)

Vous pouvez désormais installer RevLearn sur votre bureau ou smartphone de manière locale. Tous les cours sont téléchargés et accessibles sans aucune connexion internet grâce à notre Service Worker intelligent et persistant.

Catalogue de cours étendu

Le catalogue s'est formidablement enrichi avec l'introduction du grand module HTML & CSS, de JavaScript ES6+, des fondamentaux UI/UX, ainsi que d'un tout nouveau cours orienté Logique et Algorithmique.

Refonte Visuelle Minimaliste

Abandon total de l'ancien design basé sur le style portail public. Transition vers une interface de documentation experte focalisée sur l'essentiel : du texte pur, précis, à fort contraste visuel avec les polices système Inter et JetBrains Mono.

Version 1.0.0

Lancement Fondateur de RevLearn

Poussée sur le code initial de la plateforme. Mise en ligne des deux premiers fondamentaux du code de manière structurée : la Maîtrise de Git et le Terminal & Scripting Bash. Des bases saines pour continuer la progression !

Architecture et documentation rédigée dans un standard professionnel.
