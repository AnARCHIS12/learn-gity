Politique de confidentialité

RevLearn est un site statique. Cette page explique quelles données peuvent transiter techniquement lorsque vous consultez le site, quelles ressources externes sont chargées et quels choix vous gardez sur votre navigateur.

Type Information générale
Dernière mise à jour 27 mars 2026
Projet RevLearn

1. Données traitées lors de la visite

Le site ne propose pas de création de compte, de formulaire d'inscription ni de paiement. Il ne demande donc pas directement de nom, d'adresse postale ou d'autres informations personnelles de ce type.

Comme pour tout site web, des données techniques peuvent néanmoins transiter automatiquement entre votre navigateur et l'hébergement choisi pour le site: adresse IP, type de navigateur, système, date et heure de la requête, pages demandées et informations nécessaires au bon fonctionnement HTTP.

Ces données techniques relèvent du fonctionnement normal du web et peuvent être journalisées par l'hébergeur ou par certains services tiers utilisés par la page.

2. Ressources tierces chargées par le site

Le site charge actuellement des polices depuis fonts.bunny.net et une feuille de style d'icônes depuis cdnjs.cloudflare.com. Lors de ces chargements, votre navigateur contacte ces services, qui peuvent recevoir des données techniques de connexion comme l'adresse IP ou l'agent utilisateur.

Le footer du site peut également renvoyer vers GitHub pour consulter le dépôt ou signaler un problème. Si vous cliquez sur ces liens, vous quittez le site et vous êtes alors soumis aux conditions et à la politique de confidentialité du service concerné.

RevLearn n'affiche pas, dans l'état actuel du projet, d'outil publicitaire, de pixel marketing ni d'outil d'analytics explicitement intégré au code du site.

3. Stockage local et fonctionnement hors ligne

Le site enregistre un service worker afin d'améliorer le chargement et de permettre une mise en cache locale de certaines pages et ressources. Cela signifie que votre navigateur peut conserver des copies locales de contenus du site pour accélérer l'affichage ou permettre un usage limité hors ligne.

Ce stockage local est technique. Il n'est pas utilisé ici pour du ciblage publicitaire ou du profilage. En revanche, il peut conserver des ressources sur votre appareil jusqu'à ce que vous vidiez les données du site ou que le cache soit renouvelé.

4. Vos choix

Vous pouvez à tout moment effacer les données locales du site depuis les réglages de votre navigateur, désactiver le stockage local, supprimer le service worker ou bloquer certains chargements tiers si vous le souhaitez.

Si vous utilisez le site via une version déployée sur GitHub Pages ou une autre plateforme d'hébergement, vous pouvez aussi consulter les politiques propres à cet hébergeur pour comprendre la gestion de ses journaux techniques.

5. Contact

Pour toute question sur le projet ou pour signaler un problème, vous pouvez utiliser le dépôt GitHub du projet: AnARCHIS12/learn-gity.

Si vous êtes l'éditeur du site et que vous déployez cette version en production, pensez à compléter cette page si vos usages réels diffèrent de ce qui est décrit ici.
